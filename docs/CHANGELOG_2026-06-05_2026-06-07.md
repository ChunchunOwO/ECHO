# ECHO Next 26.6.7 更新日志

时间范围：2026-06-05 ~ 2026-06-07

## 一句话版本

这次更新主要围绕播放稳定性、曲库扫描响应、DSP / SRC 安全提示、流媒体体验和界面细节打磨。整体目标是：播放时更少被后台任务影响，专业音频功能更清楚，日常操作更顺手。

## 新增与增强

- 新增睡眠定时器，支持到点暂停、停止播放或退出应用，并接入托盘状态。
- 新增“我喜欢”歌曲导出能力，可按来源筛选后导出收藏内容。
- 增强流媒体专辑下载与专辑信息读取，减少进入流媒体专辑页时的卡顿感。
- SoundCloud 登录支持通过系统浏览器完成，登录流程更贴近真实桌面使用场景。
- 新增 Family 8x ECHO SRC 支持，并补充更直观的升采样标签和新手提示。
- HQPlayer / Signal Path 相关控制继续补强，专业播放链路的状态展示更完整。

## 播放与扫描

- 播放优先级继续提高：曲库扫描、维护和流媒体导入会尽量避开正在播放的关键时刻。
- 曲库扫描响应性优化，增加原生扫描器、进度反馈、后台读取和慢 IPC 诊断。
- 优化 MP3 元数据解析、扫描清理和缺失封面回填，减少扫描后封面或资料不完整的问题。
- 合并多碟专辑的 disc-number 子目录，减少同一专辑被拆成多个专辑的情况。
- Discord 状态在播放流媒体时会优先使用真实播放元数据。

## DSP 与音频安全

- DSP SRC 增加更适合新用户理解的说明和默认值，降低误开高风险设置的概率。
- 安全限幅器变为可配置，并优化提示方式，让用户更容易判断当前是保护、风险还是正常直通。
- 优化暂停状态快照，减少暂停后界面状态和实际播放状态短暂不一致的感觉。
- 插件导入和音频限幅逻辑继续收紧，提升异常情况下的稳定性。

## 界面与体验

- 加入全局布局、详情页返回、右键菜单等动效，界面切换更柔和。
- 修复歌词迷你播放器自动隐藏时的动效问题，减少闪动和位置异常。
- 历史页布局与播放控制继续优化，回看近期播放更顺手。
- 新增曲目右键菜单，并修复部分 Windows 输入场景下的操作问题。
- 标题栏品牌标识和更新提示做了轻量打磨。
- 支持竖排歌词显示，适合部分日文、中文歌词或特殊排版需求。

## 修复与稳定性

- 修复专辑墙图片加载和歌手头像刷新相关问题。
- 优化自动应用歌词匹配结果的流程，减少手动确认负担。
- 曲库维护任务改为更保守地延后执行，降低影响播放和界面响应的概率。
- 增强代理环境下的更新检查，并加入官网入口。
- 补充 Apache 2.0 许可证信息，并移除部分过期文档内容。

## 备注

- 本版本重点不是单个大功能，而是把播放、扫描、流媒体、DSP 和日常界面的很多小摩擦继续磨平。
- 如果你主要关心播放稳定性，这版最值得关注的是“播放优先”和“扫描后台化”相关改进。
- 如果你主要使用 DSP / SRC / HQPlayer，这版会更清楚地告诉你当前声音链路是否经过处理，以及处理是否可能影响原始输出。
